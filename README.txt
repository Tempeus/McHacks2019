//Create a 3d printed robotic catapult using Arduino

This is a simple software system comprising of a simple arm that can throw balls, which has a computer-controlled user interface.
In particular, the system comprises of a Java program on a computer connected, by USB, to an Arduino atmega2560 with is connected to a stepper motor driver controlling a Nema 23 Stepper Motor. 
The Java Program is meant to provie a graphical user interface for controlling the Arduino-based thrower system.
